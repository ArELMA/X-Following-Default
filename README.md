# X Following Default

X（旧Twitter）のホームタイムラインを開いた際に、**自動的に「フォロー中」タブへ切り替える** Chrome 拡張機能です。

URLが変わらない内部遷移（F5更新、アカウント切り替え、SPA再描画）にも対応し、
ユーザーの明示的操作を邪魔しない設計になっています。

---

## 特徴

* 初回ロード時に「フォロー中」へ自動切替
* F5 / Ctrl+R 更新でも確実に動作
* アカウント切り替え時も対応
* URLが変わらないXのSPA挙動に対応
* 不要な権限なし（content scriptのみ）
* 日本語UI前提

---

## 動作仕様

* 対象ページ：`https://x.com/home`
* 「ページを開いた直後のみ」フォロー中へ切り替え
* ユーザーが手動でタブを変更した後は干渉しません

※ Xの内部DOM構造に依存しているため、
将来の仕様変更により動作しなくなる可能性があります。

## インストール方法（手動）

1. このリポジトリをダウンロード、または clone
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 本フォルダを選択

---

## カスタマイズ

### 表示言語を変更したい場合

`content.js` 内の以下の定数を変更してください。

```js
const FOLLOWING_LABEL = "フォロー中";
```
